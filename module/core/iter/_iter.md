# core::iter

## 简介

该 module 包含可组合的外部迭代器。

### Organization

该模块按类型整理：

- [Traits](#traits) 是核心部分：这些特征定义了的迭代器的种类和功能。
- [Functions](#functions) 提供了创建迭代器的函数。
- [Structs](#structs) 包含模块特征各种方法的返回类型。主要关注创建这些 `struct` 的方法，而不是 `struct` 本身。
